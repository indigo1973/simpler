# A5 Runtime

## Current Status

**Temporary validation runtime only** - not production-ready.

The A5 runtime layer currently contains a temporary copy of A2A3's `host_build_graph` runtime for platform validation purposes. This will be replaced by A5-native implementations in future phases.

## Available Runtimes

### host_build_graph (Temporary)
- **Purpose**: Validate A5 platform layer functionality
- **Source**: Copied from A2A3 baseline
- **Status**: ⚠️ For validation only, not optimized for A5 hardware
